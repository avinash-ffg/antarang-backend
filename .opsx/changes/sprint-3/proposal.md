# Change Proposal: Sprint 3 - Execution and Scoring

## Intent

Implement the runtime execution environment and mathematical evaluation system. Enable students to start timed tests, persist responses, track timing, and execute server-side scoring against benchmarks.

## Scope

- Active assessment lifecycle tracking.

- Granular response recording interfaces.

- Chronological count-up time tracking components.

- Server-side domain score calculator logic.

- Multi-tier metric threshold configuration records.
