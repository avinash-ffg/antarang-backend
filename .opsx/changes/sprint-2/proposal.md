# Change Proposal: Sprint 2 - Content & Configurations

## Intent

Implement the structural metadata layer for the Antarang CAP platform. Establish the data management systems for question categories, question pools, multi-language support, version-controlled questionnaires, and active deployment configurations.

## Scope

- Question Categories & Question Bank management APIs.

- Multilingual localization support for questions and options.

- Questionnaire definition, sectioning, layout control, and snapshot versioning.

- Assessment Configuration Groups and assignment targeting (Org Unit, Cluster, User).

- Structural setup for Report Template entities to satisfy early foreign key constraints.
